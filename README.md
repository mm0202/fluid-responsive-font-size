# fluid-responsive-font-size
## 内容
ウィンドウサイズに合わせて文字サイズが自動調整されるmixin

### fluid-responsive-font-size
ウィンドウ幅の最大値と最小値、それに対応するフォントサイズの最大値と最小値を指定
### fluid-responsive-font-size-simple-ver
フォントサイズ1pxに対応するウィンドウ幅のみを指定する簡易版。フォントサイズはウィンドウサイズと等倍。
